# 종목데이터분석 서버

종목데이터분석 서버 프로그램

## Build

* PHP 5.3 or higher (5.3.3 tested)
    * curl.so
    * dom.so
    * fileinfo.so
    * gd.so
    * json.so
    * mbstring.so
    * mysql.so
    * mysqli.so
    * pdo.so
    * pdo_mysql.so
    * pdo_sqlite.so
    * phar.so
    * sqlite3.so
    * wddx.so
    * xmlreader.so
    * xmlwriter.so
    * xsl.so
    * zip.so
* Apache2 httpd server
    * mod_rewrite.so
* Mysql or MariaDB
